Mise en page : contenu + sidebar

CSS Grid avec deux colonnes (contenu principal + barre laterale)

Article principal

Ceci represente la zone de contenu principal. En CSS Grid, cette colonne occupe la majorite de la largeur (1fr) tandis que la sidebar a une largeur fixe de 280px sur les ecrans larges. Sur mobile, tout passe en une seule colonne.

La grille est definie avec .grille-contenu qui utilise display: grid et grid-template-columns avec des media queries pour s'adapter aux differentes tailles d'ecran.

Cette mise en page est tres courante pour les blogs, les sites de documentation et les pages d'articles.
